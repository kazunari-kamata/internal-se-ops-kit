# セキュリティチェックリスト

社内SEのセキュリティ確認は、日常運用で継続できる粒度に落とし込むことが重要です。すべてを一度に確認するのではなく、月次、四半期、年次の観点に分けて実施します。

## 月次確認

- 退職者、異動者、休職者のアカウント状態を確認する
- 管理者権限の付与状況を確認する
- 未適用の重要更新が残っていないか確認する
- 共有ストレージの公開範囲を確認する
- 端末台帳の未確認資産を確認する

## 四半期確認

- 主要システムのアクセスレビューを実施する
- 例外的な権限付与の期限を確認する
- バックアップの取得状況と復元手順を確認する
- インシデント記録から再発防止策の完了状況を確認する

## 年次確認

- 運用ルールと実作業の差分を確認する
- ベンダー連絡先と契約更新時期を確認する
- 重要システムの代替手順を見直す
- 教育資料や利用者向け案内を更新する

## 記録の残し方

確認した日付、確認者区分、対象範囲、結果、未完了項目、次の対応予定を残します。証跡には実データや画面キャプチャを含めず、公開用の資料ではサンプル値に置き換えます。
